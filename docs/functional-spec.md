# 기능 명세서 — SNS 포트폴리오

## 1. 회원 관리

### 1.1 회원가입
- 이메일, 비밀번호, 닉네임 입력
- 비밀번호 BCrypt 암호화 저장
- 중복 이메일 검증

### 1.2 로그인
- 이메일/비밀번호 인증
- JWT 토큰 발급 (만료기간 30일)

### 1.3 내 정보 조회
- JWT 토큰 기반 인증된 사용자 정보 반환

## 2. 게시글

### 2.1 게시글 작성
- 제목, 본문 입력
- 작성자 자동 설정 (JWT 인증)

### 2.2 게시글 조회
- 전체 게시글 목록 조회 (페이징)
- 내가 작성한 게시글 목록 조회 (페이징)
- 게시글 상세 조회

### 2.3 게시글 수정
- 본인 작성 게시글만 수정 가능
- 제목, 본문 수정

### 2.4 게시글 삭제
- 본인 작성 게시글만 삭제 가능
- Soft Delete 처리 (`removed_at` 타임스탬프 기록)

## 3. 댓글

### 3.1 댓글 작성
- 게시글에 댓글 작성
- 댓글 작성 시 게시글 작성자에게 Kafka 알림 이벤트 발행

### 3.2 댓글 조회
- 게시글별 댓글 목록 조회 (페이징)

### 3.3 댓글 수정/삭제
- 본인 작성 댓글만 수정/삭제 가능
- Soft Delete 처리

## 4. 좋아요

### 4.1 좋아요 토글
- 게시글 좋아요 등록/취소 (토글 방식)
- 중복 좋아요 방지
- 좋아요 시 게시글 작성자에게 Kafka 알림 이벤트 발행

### 4.2 좋아요 카운트
- 게시글별 좋아요 수 조회
- COUNT 쿼리 최적화

## 5. 알림

### 5.1 SSE 구독
- 로그인 사용자 SSE 연결 (실시간 알림 수신)
- ConcurrentHashMap 기반 Emitter 관리

### 5.2 알림 목록 조회
- 사용자별 알림 목록 조회
- 알림 타입: 댓글, 좋아요

## 6. Kafka 이벤트

### 6.1 알림 이벤트 발행
- 댓글 작성 시 → 게시글 작성자에게 알림
- 좋아요 시 → 게시글 작성자에게 알림

### 6.2 알림 이벤트 소비
- Kafka Consumer가 알림 이벤트 수신
- DB 저장 및 SSE를 통한 실시간 푸시
